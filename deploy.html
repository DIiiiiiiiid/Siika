硬币合成大师 - GitHub Pages发布指南

请按照以下步骤将您的游戏发布到GitHub Pages，这样您的朋友就可以通过链接直接访问并玩游戏。
1 创建GitHub账号

如果您还没有GitHub账号，请访问 https://github.com/signup 创建一个账号。

2 创建新仓库

登录后，点击右上角加号，选择"New repository"，或直接访问 https://github.com/new

  • 仓库名称输入: Siika
  • 选择"Public"公开选项
  • 点击"Create repository"按钮
3 上传您的游戏文件

在新建的仓库页面中:

  • 点击"uploading an existing file"链接
  • 将您整个项目的所有文件拖拽到上传区域(包括index.html、src文件夹、assets文件夹等)
  • 确保至少包含以下文件：
    • index.html (主页面)
    • src/game.js (游戏逻辑)
    • src/style.css (样式表)
    • assets/ 文件夹 (所有图像资源)
  • 添加提交信息，例如: "Initial game files upload"
  • 点击"Commit changes"按钮
4 启用GitHub Pages

文件上传完成后:

  • 点击仓库顶部的"Settings"选项卡
  • 在左侧菜单找到"Pages"选项
  • 在"Source"部分，将分支选择为"main"
  • 点击"Save"按钮
  • 等待几分钟，页面会显示您的网站URL
5 访问和分享您的游戏

部署完成后，您的游戏将会在以下地址可用:

https://DIiiiiiiiid.github.io/Siika/

您可以将此链接分享给您的朋友，他们点击链接就能直接玩游戏了！

注意: GitHub Pages部署通常需要几分钟时间。如果您访问链接后看到404错误，请稍等片刻再试。
